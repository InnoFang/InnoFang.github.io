项目经历

  • FamilyBond(家宝) (2017.9 ~ 2017.10)

    家宝是一款针对老人群体，主要实现子女沟通、老人监护以及健康评估等功能，满足老人和子女之间的情感和健康的需求的APP。使用组件化的方式开发，将每个功能模块分成组件，便于功能的单独开发与调试。家宝具有4个功能，老人心率检测，可疑短信拦截，地图位置共享，消息提醒。主要有三个难点，基于光电容积脉搏波描记法（PPG）的心率测量，基于NLP的可疑短信拦截，基于及时通讯的消息提醒和位置共享

  • GankIO (2017.2.7 ~ 2017.2.18)

    结合 Gank.io API 所做的第三第三方客户端，使用了 MVP + OkHttp3 + RxJava2 + Retrofit2 + Glide 的框架结构

  • 畅途 (2016.8 ~ 2016.10)

    “畅途” ———— 是一款结合针对交通安全较差的地区、安全意识较低人群以及面向出门远行人生地不熟而开发的一款服务类应用软件。具有用户位置实时显示，快速播报位置信息，拍照记录，足迹记录，轨迹显示，地图共享，紧急播报，附近等功能。我在此项目中，担任项目负责人，是学习 Android 半年后的作品，并获得了 2016 年第七届华北五省（市、自治区）及港澳台大学生计算机应用大赛全国一等奖
